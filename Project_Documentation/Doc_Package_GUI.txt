# Package: gui
This package contains the user interface code.

## File: UI.java
A custom Framework class to simplify Swing development.
### Key Methods:
- **UI(title, w, h)**: Constructor, sets up the window (JFrame).
- **createPage(name)**: Creates a basic panel and adds it to the CardLayout.
- **createSplitPage(name, leftWidth)**: Creates a Dashboard layout.
    - Uses a `JSplitPane`.
    - Left side: Menu.
    - Right side: Another `CardLayout` for content.
- **createRightCard(splitPageName, cardName)**: Adds a page to the right side of a split page.
- **showPage(name)**: Switches the main window content.
- **showRightCard(...)**: Switches the content of a split page's right side.
- **addLeftMenuButton...**: Adds buttons to the left menu.
- **applyTheme(Component)**: Applies background/foreground colors to components.
- **tableFull/table**: Helper methods to create JTables with specific data.

## File: LMSGui.java
The main application logic and screen definition. It extends `UI`.
### Fields:
- `userManager`: For login/signup.
- `dataManager`: For course/assignment data.
- `currentUser`: Stores the `User` object of the logged-in person.
- `coursesCard`, `assignmentsCard`: References to panels to update them later.
- `coursesTable`, `assignmentsTable`: References to tables to handle selection.
### Methods:
- **LMSGui()**: Constructor. Sets up Login, Signup, and Dashboard pages. Shows Login first.
- **setupLoginPage()**: Creates the Login screen with fields and buttons.
- **setupSignupPage()**: Creates the Signup screen.
- **setupDashboard()**: Creates the Dashboard skeleton (Profile, Courses, etc. placeholders).
- **handleLogin()**:
    - validation: Checks if fields are empty.
    - logic: Asks `userManager` to validate.
    - success: Sets `currentUser`, calls `updateDashboard`, shows Dashboard.
- **handleSignup()**: Creates a new Student and saves it.
- **updateDashboard(User user)**:
    - **Crucial Method**: This refreshes the whole dashboard based on who logged in.
    - Updates Profile labels.
    - Rebuilds **Courses Table**.
        - If Student: Shows enrolled courses.
        - If Admin: Shows all courses + "Add" / "Delete" buttons.
    - Calls `updateAssignmentsCard` and `updateNoticesCard`.
- **updateAssignmentsCard(User)**:
    - Rebuilds Assignment table.
    - If Admin: Adds "Add" and "Delete" buttons.
- **updateNoticesCard(User)**:
    - Rebuilds Notice table.
    - If Admin: Adds "Add" and "Delete" buttons.
- **show...Popup()**: Helper methods that show small dialog windows for entering data (e.g., "Add Course").
